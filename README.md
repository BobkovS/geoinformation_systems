Лабораторные работы по геоинформационным системам

Программный код написан на ЯП Python   

Для запуска необходимо установить в систему ПО:   
1. Anaconda for Python 3+   
   
После установки анаконды необходимо создать виртуальную среду и установить в неё библиотеку oxmnx с помощью команд:   
1. conda config --prepend channels conda-forge   
2. conda create -n ox --strict-channel-priority osmnx
3. conda activate ox

После этого можно запустить приложение из главной директории проекта:
1. Python app.py

Входящие данные приложение получает через POST запросы:
1. Для первой задачи: http://0.0.0.0:9999/town_buildings_variances с данными 
```json
{
	"town_name": "Санкт-Питербург"
}
```
